# Exercício 11 — Contador de Vogais

Crie um programa que:
- receba uma palavra;
- conte quantas vogais existem nela.

### Exemplo:
```bash
Digite uma palavra: computador

Quantidade de vogais: 4
```

# Exercício 12 — Soma de Números Pares

Crie um programa que:
- mostre a soma de todos os números pares de 1 até 100.

### Exemplo:
```bash
Resultado: 2550
```

# Exercício 13 — Login Simples

Crie um programa que:
- peça um usuário;
- peça uma senha;
- permita acesso apenas se:
  - usuário = admin
  - senha = 1234

### Exemplo:
```bash
Usuário: admin
Senha: 1234

Acesso permitido
```

---

# Exercício 14 — Maior Número da Lista

Crie um programa que:
- peça 5 números ao usuário;
- armazene em uma lista;
- mostre o maior número digitado.

### Exemplo:
```bash
Digite um número: 5
Digite um número: 8
Digite um número: 2
Digite um número: 15
Digite um número: 9

Maior número: 15
```


# Exercício 16 — Calculadora Simples

Crie um programa que:
- receba dois números;
- receba uma operação:
  - +
  - -
  - *
  - /
- execute o cálculo.

### Exemplo:
```bash
Número 1: 10
Número 2: 5
Operação: *

Resultado: 50
```

### Exemplo:

```bash
...
```

# Exercício 18 — Verificador de Senha Forte

Crie um programa que:
- receba uma senha;
- informe se ela é:
  - forte (8 caracteres ou mais)
  - ou fraca.

### Exemplo:
```bash
Digite sua senha: python123

Senha forte
```

---

# Exercício 19 — Média de Vários Alunos

Crie um programa que:
- pergunte quantas notas deseja inserir;
- receba todas as notas;
- calcule a média final.

### Exemplo:
```bash
Quantas notas? 4

Nota: 7
Nota: 8
Nota: 10
Nota: 5

Média: 7.5
```

---

# Exercício 20 — Adivinhe o Número

Crie um programa que:
- defina um número secreto;
- peça tentativas ao usuário;
- informe:
  - "Muito alto"
  - "Muito baixo"
  - ou "Acertou"

### Exemplo:
```bash
Digite um número: 7

Muito baixo
```

---

# Exercício 21 — Lista de Compras

Crie um programa que:
- permita adicionar itens em uma lista;
- exiba todos os itens ao final.

### Exemplo:
```bash
Digite um item: arroz
Digite um item: feijão
Digite um item: leite

Lista de compras:
- arroz
- feijão
- leite
```
