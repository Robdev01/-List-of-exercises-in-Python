# Exercício 1 — Soma de Dois Números

Crie um programa que:
- receba dois números do usuário;
- some os dois valores;
- mostre o resultado na tela.

### Exemplo:
```bash
Digite o primeiro número: 10
Digite o segundo número: 5

Resultado: 15
```

---

# Exercício 2 — Média de Notas

Crie um programa que:
- receba 3 notas;
- calcule a média;
- mostre o resultado.

### Exemplo:
```bash
Nota 1: 7
Nota 2: 8
Nota 3: 9

Média: 8.0
```

# Exercício 7 — Sistema de Aprovação

# Exercício 10 — Tabuada

Crie um programa que:
- receba um número;
- mostre a tabuada desse número de 1 até 10.

### Exemplo:
```bash
Digite um número: 5

5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
...
5 x 10 = 50
```
